# Machi

一個即將被完成的專案。

## 開始使用

在開始使用這個專案之前，你需要先安裝必要的依賴。請在 `next` 和 `express` 的路徑下各自執行以下命令：

npm i

## 資料庫建置
1. 在本機的 mysql 建立一個 machi_big_db 資料庫  
2. 匯入 Machi 資料夾中 machi_big_db 檔案  
3. 新增權限帳號 admin 密碼 12345
